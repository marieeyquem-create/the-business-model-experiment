Comprendre le modèle

La logique économique à piloter

Dans un modèle de location, la rentabilité dépend de votre capacité à faire travailler un actif limité : une salle vide, un véhicule immobilisé ou du matériel non loué coûtent encore de l'argent. Le chiffre d'affaires doit donc être lu avec le taux d'occupation, le revenu par unité disponible, les coûts d'usage, les charges fixes et les investissements à financer. Les indicateurs clés sont le taux d'occupation, le revenu par unité disponible, la marge après coûts variables, le point mort d'occupation, le profit opérationnel et la trésorerie.

Présentation

Un modèle conçu pour votre réalité économique

Ce template modélise une activité de location ou de mise à disposition d'actifs : salles, studios, véhicules, équipements, machines, box, espaces, matériel événementiel ou postes de travail. Vous paramétrez les actifs, les formules de location, les volumes loués mois par mois, les coûts variables, les charges fixes et les investissements. Le fichier calcule automatiquement le chiffre d'affaires, l'occupation, la marge, le point mort, le profit et la trésorerie sur 24 mois.

Guide pédagogique

Les décisions expliquées pas à pas

Actif loué

Un actif est ce que vous mettez à disposition : salle, studio, véhicule, matériel, machine, box, emplacement ou équipement.

Capacité disponible

La capacité correspond au nombre de créneaux, jours, heures ou mois réellement disponibles à la location.

Taux d'occupation

Le taux d'occupation compare ce qui est effectivement loué à la capacité disponible. C'est souvent le levier central de rentabilité.

Revenu par unité disponible

Cet indicateur montre combien chaque unité de capacité rapporte réellement, même quand elle n'est pas toujours louée.

Coûts variables

Ce sont les coûts déclenchés par l'usage : nettoyage, consommables, maintenance liée à l'utilisation, commission de plateforme ou préparation.

Point mort d'occupation

Le point mort indique le niveau minimal d'occupation nécessaire pour couvrir les charges fixes imputées à l'activité.

Investissements

Certains actifs nécessitent un achat, un aménagement ou un dépôt initial. Le template suit l'impact sur la trésorerie.

FAQ

Questions fréquentes

Puis-je modéliser plusieurs types d'actifs ?

Oui. La feuille Actifs permet de distinguer plusieurs catégories : salle, studio, véhicule, matériel, box ou équipement.

Le template fonctionne-t-il si je loue à l'heure et à la journée ?

Oui. Les formules de location permettent de saisir différentes unités : heure, demi-journée, journée, mois ou forfait.

Pourquoi suivre le taux d'occupation ?

Parce qu'un actif peu loué continue souvent à générer des charges fixes. Le taux d'occupation permet de savoir si la capacité disponible travaille suffisamment.

Les investissements sont-ils pris en compte ?

Oui. Les achats, aménagements ou dépôts initiaux peuvent être saisis dans Investissements pour analyser leur impact sur la trésorerie.
